# 美食推荐与探店分享平台 - 前端

## 技术栈
- Vue 2.6
- Vue Router
- Vuex
- Element UI
- Axios

## 安装依赖
```bash
npm install
```

## 运行项目
```bash
npm run serve
```

## 构建项目
```bash
npm run build
```

## 项目结构
- src/api - API接口
- src/components - 组件
- src/layouts - 布局
- src/router - 路由配置
- src/store - 状态管理
- src/views - 页面视图
